# SipLib.Video.Windows Class Library Documentation
This directory contains the GitHub Pages documentation project for the SipLib.Video.Windows class library.

This is a [DocFX](https://dotnet.github.io/docfx) project so DocFX must be installed on you system before you can build the documentation GitHub Pages static web site for the SipLib.Video.Windows project.

To build the documentation, open a command prompt and change to the the SipLib.Video.Windows directory and then type docfx.
